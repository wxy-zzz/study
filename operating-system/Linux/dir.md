# file path

- `/`: root
  - `/bin`: (user) binary
  - `/boot`:boot loader
  - `/dev`: device
  - `/etc`: configuration
  - `/home`: home
  - `/lib`: system library
  - `/mnt`: mount directory
  - `/proc`: process information
  - `/sbin`: system binary
  - `/srv`: service
  - `/tmp`: temporary
  - `/usr`: user system resource
  - `/var`: variable
