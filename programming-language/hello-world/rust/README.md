# Rust
https://www.rust-lang.org

## Install
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Generate project
```bash
cargo new rust (this dir)
```

## run
```bash
$ cd rust
$ cargo run
   Compiling rust v0.1.0 (~~~/rust)
    Finished dev [unoptimized + debuginfo] target(s) in 4.02s
     Running `target/debug/rust`
Hello, world!
```
